## Quick Start Notes

### For [development](docs/development/local-development.md)

1.  Run `yarn`
2.  In one terminal
3.  Run `yarn chain`
4.  In another terminal
5.  Run `yarn deploy`
6.  Run `yarn dev`

### For [publishing](docs/development/publish-npm.md)

1.  Run `yarn update-version`
2.  Then run `yarn publish-latest`
